Création de compte Active Directory

La création de compte Active Directory est une tâche récurrente dans une entreprise.
Elle permet de créer un compte utilisateur sur le domaine de l'entreprise.
Ce compte permet à l'utilisateur de se connecter à son poste de travail et d'accéder aux ressources partagées sur le réseau.

Notamment :

  • Les dossiers partagés
  • Les imprimantes
  • Les applications métiers (MyReport,Sage)
  • VPN Windows

Comment créer un compte sur l'Active Directory

La création d'un compte sur l'Active Directory se fait via l'outil "Utilisateurs et ordinateurs Active Directory".
Il suffit de faire un clic droit sur l'OU (Organizational Unit) où l'on souhaite créer le compte et de cliquer sur "Nouveau" puis "Utilisateur".

Il faut ensuite renseigner les informations du compte utilisateur :

  • Nom de connexion
  • Nom complet
  • Mot de passe
  • Expiration du mot de passe
  • Adresse mail
  • Numéro de téléphone
  • Adresse
  • ...

Une fois le compte créé, il faut l'ajouter au groupe de sécurité correspondant à son poste de travail.

Il faut ensuite ajouter le compte à l'OU correspondant à son poste de travail.

Gestion des droits serveur de fichier par groupe

La gestion des droits serveur de fichier par groupe permet de donner des droits d'accès à des dossiers partagés à des groupes d'utilisateurs.

Il faut pour cela créer un dossier partagé sur le serveur de fichier.

Il faut ensuite ajouter le groupe de sécurité correspondant aux utilisateurs qui doivent avoir accès au dossier partagé.

Il faut ensuite donner les droits d'accès au groupe de sécurité sur le dossier partagé.

Déverouillage sessions utilisateur

Le déverouillage de session utilisateur permet de débloquer un compte utilisateur qui a été bloqué suite à un trop grand nombre de tentatives de connexion.

Il faut pour cela se connecter à l'Active Directory et rechercher le compte utilisateur.

Il faut ensuite faire un clic droit sur le compte utilisateur et cliquer sur "Déverouiller le compte".
